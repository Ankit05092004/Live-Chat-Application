# Live Chat Application

A real-time, bi-directional live chat application built using **Spring Boot**, **WebSockets**, **STOMP**, and a responsive frontend powered by **Thymeleaf**. This application demonstrates how to handle real-time message broadcasting and user presence management (join/leave events) efficiently.

---

## 🚀 Features

* **Real-Time Messaging:** Instantaneous message delivery using WebSockets and STOMP messaging protocol.
* **User Presence Tracking:** Automatically notifies the chatroom when a user joins or leaves.
* **Dynamic UI:** Built with Thymeleaf templates and styled seamlessly for a clean user experience.
* **Broadcasting Architecture:** Utilizes an in-memory message broker to route messages to all connected clients.

---

## 🛠️ Tech Stack

* **Backend:** Spring Boot v3.x, Spring WebSocket
* **Frontend:** Thymeleaf, HTML5, CSS3, JavaScript
* **Protocols:** WebSocket, STOMP
* **Dependencies:** SockJS Client, StompJS

---

## ⚙️ Prerequisites

Before running the project locally, ensure you have the following installed:
* **Java 17** or higher
* **Maven 3.6+**
